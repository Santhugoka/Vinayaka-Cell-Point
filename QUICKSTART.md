## ⚙️ Quick Customization

### Change Phone Numbers
**Find and Replace in ALL files:**
- Old: `8142961113` → New: `YOUR_PHONE_1`
- Old: `9392166166` → New: `YOUR_PHONE_2`
- WhatsApp: `918142961113` → `91YOUR_PHONE`

**Files to edit:** All `.html` files

### Update Business Hours
**File:** `location.html`
```html
Find: "9:00 AM - 8:00 PM"
Replace with: "YOUR_HOURS"
```

### Change Shop Location
**File:** `location.html`
- Find Google Maps link and iframe
- Replace with your location URL from Google Maps

### Add New Service
**File:** `services.html`
- Copy a `<div class="service-item">` block
- Update icon, title, and description
- Save and refresh browser

### Update Offers
**File:** `offers.html`
- Edit offer cards with new details
- Change offer badge text
- Update descriptions and buttons

## 🔧 How to Edit Files

### Using Notepad/Text Editor
1. Right-click on any `.html` or `.css` file
2. Select "Open with" → "Notepad" or your editor
3. Make changes
4. Save file (Ctrl+S)
5. Refresh browser to see changes

### Using VS Code (Recommended)
1. Install VS Code from https://code.visualstudio.com
2. Drag and drop the "Mani 2" folder into VS Code
3. Edit files directly in the editor
4. Save changes
5. Browser auto-refreshes (with Live Server extension)

## 🐛 Troubleshooting

### Pages not opening
- ✅ Make sure all files are in the same folder
- ✅ Check file names are spelled correctly
- ✅ Use .html extension for all page files

### Buttons not working
- ✅ Check phone numbers are correct
- ✅ Ensure links start with `tel:` or `https://wa.me/`
- ✅ Verify no spaces in URLs

### Design looks broken
- ✅ Clear browser cache (Ctrl+Shift+Del)
- ✅ Try different browser
- ✅ Check internet connection
- ✅ Verify style.css is in same folder

### Mobile menu not working
- ✅ Ensure script.js is in same folder
- ✅ Check JavaScript is enabled in browser
- ✅ Resize browser window to test
